Stratix II EP2S60 DSP Board
The Stratix II EP2S60 DSP board is a development platform for high-performance digital signal processing (DSP) designs, and features the Stratix II EP2S60 device in a 1020-pin package.
The Stratix II EP2S60 DSP board supports alternative EP2S60F1020C4 and EP2S60F1020C4ES devices, which you can select in the configuration block properties.
Table 24–8 lists the blocks available to support the Stratix EP2S60 DSP board.
 
Table 24–8. Stratix EP2S60 DSP Board Blocks 
Block
Description
A2D_1 and A2D_2
Controls the 12-bit signed analog-to-digital converters (U1, U2). You can optionally specify the clock signal.
D2A_1 and D2A_2
Controls the 14-bit unsigned digital-to-analog converters (U14, U15)
IO_DEV_CLRn
Controls the board reset push-button switch (SW8). You can optionally specify the clock signal.
LED0–LED7
Controls eight user-definable LEDs (D1–D8).
PROTO and PROTO1
Santa Cruz connectors, which controls the prototyping area I/O. You can optionally specify Input or Output node type, specify the input clock signal, and specify the pin locations (J23– J25, J26–J28).
PROTO2
Mictor connector, which controls the debugging port. You can optionally specify Input or Output node type, specify the input clock signal, and specify the pin location for each port (J20).
PROTO3
External analog-to-digital converter interface connector. You can optionally specify Input or Output node type, specify the input clock signal, and specify the pin location for each port (J5, J6).
Display0 and Display1
Controls a dual user-definable seven-segment LED display (U12, U13).
SW4–SW7
Controls four user-definable push-button switches (SW4–SW7). You can optionally specify the clock signal.
For information about setting up the board, refer to the DSP Development Kit Getting Started User Guide. For information about the supported hardware features, refer to the Stratix II DSP Development Board Reference Manual.
