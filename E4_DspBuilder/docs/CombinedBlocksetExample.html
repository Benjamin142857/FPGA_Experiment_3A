Section II. DSP Builder Standard Blockset Libraries : 26. Design Examples : Combined Blockset Example
Combined Blockset Example
Open this model
This design example illustrates how to embed a DSP Builder Advanced Blockset design inside a top-level standard blockset design. The resulting system comprises blocks from both blocksets, simulates cycle-accurately and you can us the standard blockset TestBench block to test it.
The example model is demo_adapted_ad9856.mdl.
For more information about this design example, refer to the DSP Builder chapter in the DSP Design Flow User Guide.
