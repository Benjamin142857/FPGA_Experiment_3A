Section II. DSP Builder Standard Blockset Libraries : 26. Design Examples : Avalon-MM FIFO
Avalon-MM FIFO
Open this model
This design example consists of a Prewitt edge detector with one Avalon-MM Write FIFO buffer and one Avalon-MM Read FIFO buffer. DSP Bu idler uses an additional slave port as a control port. You can include the design as an SOPC Builder peripheral to the Avalon-MM bus.
The example model is sopc_edge_detector.mdl.
For more information about this design, refer to Avalon-MM FIFO Buffer in the Using the Interfaces Library chapter in the DSP Builder Standard Blockset User Guide section in volume 2 of the DSP Builder Handbook.
