Stratix IV GX FPGA Development Board
Table 11–4 lists the blocks available to support the Stratix IV FPGA Development Board.
 
Table 11–4. Stratix IV GX FPGA Development Board Blocks  
Block
Description
Dip Switch
Controls the user-definable dual in-line package switch (SW3). You can optionally specify the clock signal.
LED0–LED15
Controls sixteen user-definable LEDs.
PB0–PB2, CPU_RESETn
Controls three user-definable push-button switches and the CPU reset push button. You can optionally specify the clock signal.
f
